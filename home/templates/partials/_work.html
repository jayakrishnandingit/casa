Sitiata

Sitiata is a complete solution for tracking billable hours, organizing tasks and billing the work done. The project was developed using Python2.7, Django1.11, PostgreSQL9.1 and jQuery. Some of the major contributions I made are,

  • Revamped time tracking functionality which significantly reduced bugs and increased reliability.
  • Implemented a Kanban board with drag 'n' drop from ground up and provided an easier interface to manage tasks.
  • Integrated AWS S3 to store user uploaded images that helped reduce server load considerably.
  • Added static file caching using CloudFlare service which substantially reduced page load latency.
  • Implemented an invoicing service which along with Reports increased the efficiency of billing.
